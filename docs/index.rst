Welcome to Googlicious's documentation!
=======================================

Googlicious wraps PyDrive to label Google docs content.

Project Info
============

- Homepage: `https://https://gnomon.securitystandard.org <https://gnomon.securitystandard.org>`_

How to install
==============

To install the current development version from GitHub, use:

::

    $ pip install git+https://github.com/GnomonConquest/googlicious.git#egg=googlicious
